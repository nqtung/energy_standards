WITSML v1.4.1.1 API files

The normative web service specification consists of the following files in the zip:

  • WITSML_STORE_API_Version_1.4.1.1.pdf defines the normative behavior of the STORE web service.
  • WMLS.WSDL is the normative definition of the STORE web service interface data structures.
  • ./xsd_schemas/*.xsd defines the normative API specific schemas that represent the content of parameters in the WSDLs.

The following files in the zip are informative:

  • ./CHANGE_DETAIL_API.TXT lists the schema changes since the previous version.
  • ./index_witsml_api.html (this file)
  • ./doc/witsml_api_overview.html provides a browsable overview of the specification.
  • ./doc/schema/*.html represents a browsable view of the XSD files.
  • ./stylesheets/*.xsl are stylesheets that can be used to view the examples.
  • ./XML_Examples/*.xml are examples of each object.
